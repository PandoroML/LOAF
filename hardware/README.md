# LOAF Hardware

Open source hardware for hyperlocal weather monitoring with no vendor lock-in.

## Components

### Ultrasonic Anemometer

DIY ultrasonic wind sensor for measuring wind speed and direction.

See the [anemometers/](anemometers/) directory for designs and build instructions.

### Data Logger

- **Board**: Raspberry Pi (3B+ or newer recommended)
- **Interface**: RS-485/SDI-12 for sensor communication
- **Enclosure**: 3D printed weatherproof housing

### Power System

For remote deployment:

- Solar panel (size depends on location and power requirements)
- Battery backup for continuous operation
- Charge controller
